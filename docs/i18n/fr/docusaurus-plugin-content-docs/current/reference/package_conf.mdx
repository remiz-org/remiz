# Spécification du fichier de configuration de package

## Emplacement du fichier
Le nom et l'emplacement du fichier ne sont pas vraiment important.
Le fichier est seulement utilisé lors de la construction du package, il faut
juste être sûr que le chemin et le nom du fichier sont corrects lors de
l'appel à la commande `remiz build`.

## Exemple
```json title="foo.toml"
[info]
name = "My project"
description = "My project package description"
version = "1.2.3"

[copy_files]
test_1 = "./src/test_1.sh"
test_2 = "./src/test_2.py"
test_3 = [
    "./data/1.csv",
    "./data/2.csv",
    "./data/3.csv",
]

[bar]
bar_1 = "./src/test_1.sh"
```
