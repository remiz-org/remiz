# Spécification du fichier de configuration globale

## Emplacement du fichier
Il est recommandé de positionner le fichier `configuration.toml` dans le même
répertoire que le binaire Remiz. Il est tout de même possible de le positionner
ailleurs, l'appel à Remiz devra se faire par l'ajout du flag `-g` permettant
de spécifier son emplacement.

## Exemple de fichier de configuration
```json title="configuration.toml"
[packagers]
copy_files = "packagers/copy_files.py"
bar = "packagers/bar.py"

[store]
local = "packages/{name}/{name}_v{version}.pack"
```
